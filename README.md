# acs_8000_skill

This repository contains a Warp custom skill for helping users operate the Vertiv/Avocent ACS800 and ACS8000 Advanced Console System CLI.

## What's in this repo
- `.agents/skills/acs8000-cli/SKILL.md`
  - Main skill instructions for CLI navigation, configuration workflows, and operational recipes.
- `.agents/skills/acs8000-cli/references/core-commands.md`
  - Detailed command and administration reference aligned to the guide sections.
- `.agents/skills/acs8000-cli/references/section-index.md`
  - Section index to quickly jump to the right part of the manual.
- `.agents/skills/acs8000-cli/references/acs8000-command-reference-transcript.md`
  - Near-complete text transcript extracted from the source PDF for deep/edge-case lookups.

## Source PDF used to compile this skill
- Vertiv command reference guide:
  - https://www.vertiv.com/48eeed/globalassets/shared/avocent-acs-800acs-8000-advanced-console-system-command-reference-guide.pdf
